# FreeSql 入门

## 安装

FreeSql 是一个 .NET Standard 2.0 库，支持 .NET Framework 4.6.1 或 .NET Core 或更高版本的应用程序。

```shell
dotnet add package FreeSql
```

或者

```shell
Install-Package FreeSql
```

## 入门教程

FreeSql 可基于现有数据库创建模型，也可基于模型创建数据库。 提供的教程演示了这两种方法。
